## Part 1 第一次课堂笔记：算法(algorithm)和模型(model)
### 算法(algorithm)：
- **定义**：用于解决特定问题或执行特定任务的一系列明确步骤或规则
- **特点**：过程导向，描述“如何做”

### 模型(model)：
- **定义**：通过算法从数据中学习得到的结果
- **特点**：结果导向，代表“学到了什么”

### 算法(algorithm)和模型(model)的区别与联系：
- **核心区别总结**：

| 对比维度 | 算法 | 模型 |
|:----------|:------|:------|
| **本质** | 过程与方法 | 结果与表示 |
| **关注点** | 如何学习/计算 | 什么被学习 |
| **目标** | 高效、正确地解决问题 | 准确地抽象和预测现实 |

- **关键联系**：

  **算法是实现模型训练和预测的基础**，定义了学习的规则和过程；**模型则是算法在实践中的具体表现形式**，通过选择不同的算法可以获得不同类型的模型，在解决不同的问题时算法的选择对模型的效果至关重要。
